# Домашнее задание lab04

Лабораторная работа Лисейчиковой Екатерины

## Что требовалось сделать

Домашнее задание относится к теме лабораторной работы IV: изучение систем непрерывной интеграции на примере travis ci.

## Выполненные пункты

- для Linux используется матрица компиляторов `gcc` и `clang` в `.travis.yml`; 
- для Windows используется `.appveyor.yml` с генератором `Visual Studio 17 2022`; 
- оба сценария запускают CMake одинаковым способом, поэтому поведение сборки остается переносимым;
- конфигурации не содержат персональных токенов или локальных путей.

## Как проверить домашнее задание

```bash
cmake -S . -B _build -DCMAKE_BUILD_TYPE=Release -DBUILD_TESTS=ON
cmake --build _build
ctest --test-dir _build --output-on-failure
cmake --install _build --prefix _install
```

## Пояснение по реализации

Проект оформлен так, чтобы его можно было перенести в Linux-окружение и собрать стандартными командами CMake. Файлы не завязаны на абсолютные пути macOS, персональные секреты не добавлены в git, а дополнительные инструменты подключаются через конфигурационные файлы конкретной лабораторной.
